# 🥘 REcipify Beta v1.0 — Релізний опис

Вітаємо у першій бета-версії **REcipify**! Це розумний помічник для керування рецептами та відстеження харчування, що використовує можливості штучного інтелекту.

## 🚀 Що нового у цьому релізі?

### 🤖 Інтелект Groq AI
- **Автоматичний аналіз:** Додавайте інгредієнти, а наш ШІ автоматично розрахує КБЖВ (калорії, білки, жири, вуглеводи) для всієї страви.
- **Розумні підказки:** Отримуйте рекомендації щодо приготування в режимі реального часу.

### 📚 Організація та пошук
- **Система папок:** Створюйте власні категорії (Сніданки, Вечері, ПП-рецепти тощо) для зручного зберігання.
- **Швидкий пошук:** Знаходьте потрібні страви за назвою, інгредієнтами або складністю завдяки оптимізованим алгоритмам.
- **Офлайн-режим:** Всі ваші рецепти зберігаються локально (Room DB) і доступні навіть без інтернету.

### 📊 Здоров'я та прогрес
- **Дашборд активності:** Відстежуйте спожиті калорії за сьогодні у зручному графічному форматі.
- **Історія харчування:** Ведіть детальний лог своїх прийомів їжі, щоб бачити прогрес у досягненні цілей.
- **Система "Streak":** Підтримуйте серію активних днів та формуйте корисні звички.

### 📸 Зручність та дизайн
- **Інтеграція з камерою:** Швидко додавайте фото страв.
- **Темна тема:** Повна підтримка Dark Mode для комфортного використання у будь-який час доби.
- **Сучасний інтерфейс:** Лаконічний дизайн, зосереджений на контенті.

## 🛠 Як почати тестування?
1. Встановіть файл `REcipify.apk`, який знаходиться у кореневій папці проекту.
2. Для роботи функцій ШІ переконайтеся, що у вас налаштовано API-ключ у файлі `keys.properties`.
3. Створюйте папки, додавайте рецепти та діліться фідбеком!
